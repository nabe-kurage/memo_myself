## Markdown

### ・画像を表示する
```
![代替テキスト](画像のURL)

- ここ（Readme）に貼りたい時はissueに一回ドラッグアンドドロップして文字列をコピペすると早い
```



### ・リンクを入れる
```
[表示文字](URL)
```

### ・ページ内リンクを入れる
```
[hoge](#hoge)
# Hoge

- 見出しの深さに関係なく#は一つ
- 大文字は全て小文字にする
- 半角空白はハイフン(-) を使う

```
